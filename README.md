# Music_Application
This is a front and rear separation music application.
## Tech Stack
**Front End**: Vue 

**Back End**: SpringBoot, Maven, Mysql, MyBatis
## How to run this project
1. music-client
```
  npm install
  npm run dev
```
Visit on localhost://8081
2. music-manage
```
  npm install
  npm run dev
```
Visit on localhost://8080
3. music-server
Run in Idea
